## **第1章  逃离校园1**
这是最后一节课了，上完这节课，刘岄就可以和男友吴智见面了。<br />这还是两人上大学以来第一次见面，为此刘岄今天还特意起大早化了个精致的妆容。<br />见刘岄频频看向手机的时间，她的室友纷纷捂嘴笑道：“别急，再等五分钟就下课了，就这一会，你都看了好几次手机了。恋爱脑要不得啊。”<br />刘岄听了也有些不好意思，只得按捺下焦躁。她没有回复室友的调笑，也没有去解释其实不是期待和男友见面，只是因为今天从睡醒起来后就有些心绪不宁，但是她也说不出来一个所以然。<br />伴随着铃声响起，老师终于结束了一下午的课程，她布置了一些作业后便准备离开教室。<br />刘岄一直看着老师，注意到她走到门口又停了下来，她感到不解，随着老师的目光一起看向了天空，这时她才注意到，外面乌云密布。<br />“奇怪了，刚刚还没变天啊。”老师喃喃自语，拎着包快步离开了，大概是希望赶在雨下起来之前回到宿舍吧。<br />“刘岄，你带了伞吗？”室友都担心的看着刘岄。<br />刘岄摇了摇头，室友何微见状说到：“要不你先和我们回宿舍吧，你这样赶去校外，路上下雨了怎么办，你男朋友也不一定带了伞。”<br />“是啊是啊，还是跟我们一起回去吧？”其他室友也搭腔。<br />刘岄见她们都这么关心自己，心里微暖。<br />“那好，我先给我男朋友发个信息，叫他等我一会。”<br />几人说完就收拾好东西一起走回宿舍，出门刚好碰到一个女生擦肩而过，一股淡淡的香味从刘岄鼻尖划过，她目光下意识追寻着走远的女生。<br />何微几人正在叽叽喳喳的讨论等下吃什么，见刘岄不知看着哪里发呆，跟着看过去：“刘岄你一直看那个女生干嘛啊？”<br />“嗯...她好漂亮啊，你们知道她是谁吗？”刘岄将目光收回，加入到室友的聊天之中。<br />“不知道欸，好像是艺术学院的吧，穿得好好看。也就她们艺术学院的小姐姐有这么好的审美吧？”<br />这话引起了剩下几个女孩子的不满，几个人打打闹闹的说着话没再关注走远的女生。<br />天色暗得非常快，几分钟之前路边不开灯还能勉强看清远处，这会儿已经要依靠手电筒的光去照路了。几人也不玩闹了，加快速度赶回了宿舍。<br />室友们看刘岄刚拿了伞就准备出门，还是忍不住多叮嘱一句：“这种天气，实在不行就别约会了，让你男朋友自己找个宾馆住下好了。”<br />刘岄无奈道：“你们这是要我放人鸽子啊，他特意从那么远过来。”<br />刘岄微微叹气，知道她们都是关心自己，但是男朋友不来见自己是因为家里条件不好，两个人见面一次的车费也不便宜，他又不愿意让自己出钱。她摇了摇头不再想这些，拿着伞往外面走去。<br />刘岄微微一笑，走上前去，对他轻声说到：“久等了吧。”<br />吴智傻笑着接过她手里的伞，另一只手却有些踟蹰：“也没有等很久，这个天气确实应该回去拿伞的。那我们走吧，是先去吃饭还是？”<br />刘岄只当看不懂他的举动，两人确认关系后还是第一次见面，她想要循序渐进，还没有到一见面就牵手的地步：“你先去放好行李吧，不然背着包去吃饭也挺累的。”<br />她说到这里，想起约会前室友对自己的百般叮嘱，又补充了一句：“就在我学校附近住下吧，也方便你送我回宿舍。”<br />吴智听了这话，有些失望，但是也没表现出来：“我刚刚过来看到不远处就有个宾馆，环境看起来还可以，我们就去那里吧。你上了这么久的课，肯定也饿了。我们快点收拾好就去吃饭。”<br />何微无奈地看了她一眼：“那人家都来了，总不能不见面吧。何况谁也不知道下一波疫灾什么时候来呢。咱们上学期那会天天因为这个原因出不了门，都说四五月又要爆发新的一波疫灾，他们两可不得赶在这时候见面嘛，你也少说两句。”<br />其他室友也附和：“说多了，到时候刘岄心里也不舒服。你本来是为她好，别搞得你俩关系差了。”<br />周临听前面的话本来还在反思，最后这句话一出，她立马打断：“刘岄不会是这种人的，我相信她！”<br />几人一听也笑到：“确实，是我们说错啦。咱们还是快点吃，等下回宿舍也好催她早点回来。”<br />“就是！可不能让她在外面过夜。”周临连忙低头扒饭。<br />几人全然不知校园被一种莫名的气氛逐渐包围，空气中也弥漫开不知名气味。

刘岄和吴智放好行李出来，走远离开大门范围外，路上已经伸手不见五指了。<br />再远点有一些灯光，估计是路边的小餐馆，两个人慢慢朝那边走过去。<br />“今天真的有点奇怪，这会儿跟晚上十二点一样。”刘岄感到一丝不寻常，一阵风吹过，她忍不住搓了搓手臂。<br />吴智伸手握住她的胳膊：“你冷到了吗？”<br />刘岄一愣，想了想还是没有挣开。于是吴智的手开始慢慢下滑，改为握住她的手腕。<br />他试探着将手指附上她的手背，冰凉的触感从指尖传来。<br />“吴智。”刘岄突然喊他。<br />“啊”，吴智的手停住了，下意识收了回去，“岄岄怎么了？”<br />“你看那里。”刘岄指着不远处灯光下摇摇晃晃的身影，“那...那是什么？”<br />吴智看过去，一时也不能确定：“是不是有人喝醉了？”<br />刘岄忽然开始心跳加速，一天中的不安此刻达到顶峰：“要不我们别过去了吧？我总感觉不是那么简单。”<br />“岄岄你胆子怎么这么小了？以前放学我们走黑巷子你都不怕的啊。”吴智笑着看向刘岄，想要伸手揉一下她的脑袋，却看到她脸色发白，不由急道，“岄岄？你这是怎么了？”<br />远处人影似乎注意到这边的动静，开始一晃一晃地走过来，其中还伴随着轻轻的嘶吼声。<br />“嗬— 嗬—”<br />这下连吴智也觉得不对了，这个年代大家基本都看过生化危机。<br />刘岄看着慢慢挪步过来的身影，脑子里不断将其和电影里的画面重合，她伸手拽了拽旁边人的袖子，轻声喝道：“跑！”<br />吴智一听连忙转身往宾馆那边跑，刘岄快步跟上，也不回头看那个奇怪的人是否跟上，不敢有一丝停留。<br />宾馆的大门依旧打开着，前台却不知所踪。<br />刘岄到宾馆的第一件事就是将大门关上，避免那个奇怪的人跟进宾馆。<br />她把门关上后终于松出一口气，这时才敢往远处瞥去，那里依旧是黑漆漆一片，除了灯光范围内，一概看不清楚。<br />“岄岄，我们去房间里吧？”吴智这会也冷静下来了，他轻咳一声，“刚刚那个也不知道是什么，我们去房间会安全点？回去看看新闻，查查看这到底是什么情况。如果真有……应该会有新闻报道的，我们也别自己吓自己。”<br />刘岄知道他没说完的话是什么。<br />是丧尸吗？真的会有这种东西吗？她忍不住想到，去年五六月份的时候总在某个社交软件上刷到丧尸相关视频，说什么明年会爆发丧尸病毒。那时候她只觉得是天方夜谭，现在一时半会不知道该不该信了，万一只是恶作剧呢？可是自己这一天说不出原因的不安又是什么呢？<br />刘岄百思不得其解，只能暂时跟着他一起回到房间。<br />两人一到房间立刻将门锁死。<br />刘岄不放心，又搬来凳子堵住门。<br />吴智掏出手机，第一时间打开新闻相关内容，并没有看到任何报道。<br />刘岄则是登陆了校园网站，手指轻轻翻着网站上的帖子。忽然她像是发现了什么，停下不断划动的手指，认真看了一眼帖子名字。<br />【注意！文学院已有不少同学忽染重病，头疼脑热，浑身抽搐！】<br />刘岄吸了一口气，点开帖子。<br />【楼主：我们文学院好几个寝室的人都忽然高温不退，还伴随抽搐。学校也不处理，说就是普通感冒，不会传染。我真不放心，唉大家还是戴好口罩吧。<br />二楼：啥情况？不传染怎么会几个寝室都这样。<br />回复：不知道是不是传染的，反正差不多同一个时间都这样了，但是两三天了，同寝室没症状的人还是生龙活虎的，所以校医来过后说不会传染，叫我们自己好好注意好就行……<br />……<br />最新更新两条：<br />楼主：我总感觉不太对啊……我室友也中招了，他不仅抽搐，还老是表情很狰狞，怪吓人的。<br />楼主：救命！救命！有没有人在看帖子？我室友发疯了要咬我！我躲进厕所了！我已经打电话通知班导了，但是我没有宿管电话，宿舍也就剩我一个人，我控制不了我室友。现在有没有人能帮我！没有人帮我我就报警了啊！】

## **第2章  逃离校园2**
刘岄把手机递给吴智：“你看这个，我觉得这件事......”她话没说完，实在不知道用什么语言去形容此刻的心情。<br />吴智快速浏览完帖子，目光停留在那个帖子里楼主最新更新的内容。<br />“......有可能是巧合吧？应该不会这么魔幻吧......”他迟疑道。<br />刘岄明白这件事有多么匪夷所思，就是她自己也不敢去相信，但是不怕一万就怕万一。她也不想和吴智继续纠结这回事，把手机拿回来直接在宿舍群里发起了群通话。<br />电话很快被接通，传来女孩子们嬉戏打闹的声音。<br />“别吵了别吵了，快听一下刘岄要说什么。”有女生说到。<br />刘岄一时有些啼笑皆非，但她没忘记正事：“不是这回事，我现在分享一个校园网站的帖子给你们看，你们不要不当一回事，知道吗？我刚刚在路上已经遇到了一个很奇怪的人了。”<br />周临听着一边打开手机里的链接，和大家凑在一起看了起来：“你说的奇怪的人是什么啊？”<br
